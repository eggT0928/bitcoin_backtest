# BTC 이동평균 분할진입 + 개선 매도 백테스트

비트코인 이동평균 전략을 표와 그래프로 확인하는 Streamlit 대시보드입니다.

## 전략 규칙

### 매수, 원안 유지

| 조건 | 투자 비중 |
|---|---:|
| 5일선이 120일선 상향 돌파 | 50% |
| 25일선이 120일선 상향 돌파 | 75% |
| 99일선이 120일선 상향 돌파 | 100% |

### 원안 매도

| 조건 | 투자 비중 |
|---|---:|
| 5일선이 120일선 하향 돌파 | 0% |

### 개선 매도

| 조건 | 투자 비중 |
|---|---:|
| 5일선이 120일선 하향 돌파 | 50%로 축소 |
| 25일선이 120일선 하향 돌파 | 25%로 축소 |
| 99일선이 120일선 하향 돌파 | 0%, 전량 매도 |

## GitHub Codespaces에서 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 로컬 PC에서 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

실행 후 브라우저에 표, 누적 수익률 그래프, MDD 그래프, 비중 그래프가 표시됩니다.
